IMAGE REPLACEMENT GUIDE
=======================
All images on the site currently use https://placehold.co/ placeholders so
the site displays properly on day one. Replace each URL with a link to your
own photo (or a local file inside this /assets/images/ folder) at the
dimensions listed below for best results.

Photos should be JPG or WebP, under 300KB each, color-corrected, and
horizontally oriented unless noted otherwise.

----------------------------------------------------------------

- "Why Choose Us" right image        800  x 600    Technician mid-service, smiling team, or branded truck

- Blog preview 1                     1200 x 630    Reuse from blog/post-1.html
